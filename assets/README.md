# NetBird brand assets

`netbird-logomark.svg` and `netbird-logomark.png` are the unmodified official
NetBird Logomark downloads from the [NetBird Press Kit](https://netbird.io/press)
(`https://netbird.io/press/netbird-bird-only.svg` and
`https://netbird.io/press/NetBird_Icon.png`, retrieved 2026-07-18).

The Press Kit requires use of its official logos and says not to alter their
colors, proportions, or apply effects. `ICON.PNG`, `ICON_256.PNG`, and the
fnOS UI PNG files are high-quality resizes of the official transparent PNG;
they preserve its square aspect ratio, alpha channel, colors, and artwork.
No runtime build or package step downloads these files.

The upstream NetBird repository is BSD-3-Clause except for the directories it
identifies as AGPLv3; see its [LICENSE](https://github.com/netbirdio/netbird/blob/main/LICENSE).
This package keeps this attribution and follows the separate Press Kit brand
usage guidance for the logo artwork.
